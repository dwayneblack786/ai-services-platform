# Documentation Index

Platform documentation in this folder is now organized by ownership and lifecycle.

## Structure

- `docs/platform/` - Stable, platform-wide architecture and standards
- `docs/archive/` - Historical phase/status/completion reports
- `docs/*.md` - Active domain/topic docs that are still being classified

- `docs/archive/`
