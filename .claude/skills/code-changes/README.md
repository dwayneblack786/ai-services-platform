# Code Change Skills

Use these skills when implementing code, not only reviewing it.

Skills:

1. `senior-frontend-react-ui-node.md`
2. `senior-backend-node-architecture.md`
3. `senior-java-spring-standards.md`
4. `senior-ai-agentic-implementation.md` — AI integrations, agentic workflows, RAG, provider abstraction
5. `senior-tdd-test-first.md` — Test-Driven Development cycle for all tiers

Quick selector:

- UI-heavy React page/component work: use `senior-frontend-react-ui-node.md`
- API/service/data-layer implementation: use `senior-backend-node-architecture.md`
- Listing service, orchestration, Java backend work: use `senior-java-spring-standards.md`
- LLM integration, RAG, agent loops, model-provider abstraction, or AI orchestration: use `senior-ai-agentic-implementation.md`
- New feature with test-first approach (any tier): use `senior-tdd-test-first.md`
